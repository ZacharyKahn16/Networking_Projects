
FastFtp Driver

Running the driver:
==============
The following scripts can be used to compile and run the driver

	./build.sh --> to build the driver

	./run.sh --> to run the driver

The following options can be used to pass command line arguments to the FastFtp client by modifying the "run" script:

	-s	Host name where the FastFtp server is running.

	-p 	Server port number.

	-w 	Client window size in segments.

	-t 	Duration of the time-out- period.

	-f 	Name of the file to be transmitted to the server.
